Документация проекта: Dimension Shift Ultimate
Dimension Shift Ultimate — это двухмерный пазл-платформер на языке Python с использованием библиотеки Pygame. Основной особенностью игры является механика переключения между двумя мирами (техногенным пространством "Cyber" и органическим пространством "Fantasy"), в каждом из которых изменяется физическая структура платформ, типы препятствий и визуальное оформление.

Основные технические особенности
Процедурный синтез звука. В игре отсутствуют внешние аудиофайлы. Звуковые эффекты и фоновые треки генерируются математически непосредственно в оперативную память при запуске программы. Переход между мирами сопровождается плавным стереофоническим кроссфейдом.
Физический движок. Включает в себя механики Coyote Time (возможность прыгнуть в течение первых 8 кадров падения с уступа), Jump Buffering (буферизация ввода прыжка на 8 кадров вперед), Squash and Stretch (динамическое масштабирование спрайта при прыжках и падениях), а также инерцию движущихся платформ.
Отказоустойчивое сохранение данных. Прогресс прохождения уровней и пользовательские настройки сохраняются в JSON-формате, закодированном алгоритмом Base64. Запись выполняется атомарно через создание временного файла во избежание порчи данных при внезапном сбое процесса. Реализовано автоматическое восстановление из резервной копии (.bak).
Экран настроек. Полноценное интерактивное меню с регулировкой громкости и возможностью динамического переназначения клавиш управления.
Порядок запуска
Запуск скомпилированной версии (EXE):
Запустить файл DimensionShift.exe двойным кликом.
Запуск из исходного кода:
Установить библиотеки: pip install pygame
Запустить файл: python main.py
Компиляция проекта
Сборка выполняется с помощью утилиты PyInstaller:
Установить компилятор: pip install pyinstaller
Выполнить сборку: pyinstaller --onefile --noconsole --name="DimensionShift" main.py
Готовый файл DimensionShift.exe будет находиться в директории dist.
Системные требования
Операционная система: Windows 7 SP1 / 8 / 8.1 / 10 / 11
Процессор: x86/x64 с частотой от 1.0 ГГц
Оперативная память: от 512 МБ
Видеокарта: любая с поддержкой OpenGL 2.0 или DirectX 9.0c
Свободное место на диске: 15 МБ
Учебный проект, выполненный в рамках учебной практики ГУАП (Санкт-Петербург, 2026 г.) на кафедре Информационных систем и технологий №42.
