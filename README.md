
# Dimension Shift Ultimate 🌌

Двухмерный пазл-платформер на Python + Pygame. Главная фишка игры — прохождение уровней завязано на мгновенном переключении между двумя измерениями: техногенным неоновым киберпанком (Мир А) и уютным фэнтези-лесом (Мир B). 

Платформы, опасности и враги материализуются только в своем измерении, поэтому прыжки и траекторию движения приходится планировать прямо в воздухе.

---

## 🎯 Что здесь крутого (технические фишки)

*   **Полностью процедурный звук и генеративная музыка.** В игре **нет внешних аудиофайлов** (`.wav`, `.mp3` и др.). Все звуки прыжков, телепортации, сбора монет, а также оба бесконечных фоновых саундтрека синтезируются математически (генерацией PCM-сигнала) прямо в оперативную память при запуске. Музыка плавно перетекает одна в другую (стерео-кроссфейд) при смене миров.
*   **Плавная и «сочная» физика.** Реализованы классические консольные механики:
    *   *Coyote Time* — можно прыгнуть в течение 8 кадров после схода с платформы.
    *   *Jump Buffer* — прыжок сработает, даже если нажать кнопку за 8 кадров до приземления.
    *   *Squash & Stretch* — спрайт персонажа динамически сжимается и вытягивается при прыжках и жестких падениях.
    *   *Инерция платформ* — игрок полностью наследует скорость движущихся платформ при прыжке с них.
*   **Умное и отказоустойчивое сохранение.** Игра сохраняет рекорды по времени и монетам для каждого из 10 уровней в `highscore.json`. Файлы обфусцируются через Base64 (защита от редактирования блокнотом) и автоматически восстанавливаются из бэкапа (`highscore.bak`), если основной файл повредился или запись прервалась.
*   **Экран настроек и переназначение клавиш.** В игре есть рабочее меню настроек, где можно отрегулировать громкость ползунками или переназначить любую клавишу под себя прямо на экране с мгновенным сохранением конфига.

## 🚀 Как запустить

### Быстрый запуск (готовый EXE)
Если вы скачали скомпилированную версию:
1. Запустите файл `DimensionShift.exe`.
2. Играйте (установка Python и библиотек не требуется).

### Запуск из исходного кода
Если хотите запустить проект через консоль или посмотреть код:
1. Установите Python 3.10+ и библиотеку Pygame:
   ```bash
   pip install pygame

2.  Переименуйте исходный файл в main.py и запустите его:
    python main.py

🛠 Как собрать свой EXE

Поскольку в проекте полностью отсутствуют внешние ресурсы (картинки, музыка,
звуки), его чрезвычайно просто компилировать в один исполняемый файл без
необходимости прописывать пути к ассетам.

Сборка выполняется с помощью утилиты PyInstaller:

1.  Установите сборщик:
    pip install pyinstaller
2.  Соберите проект одной командой:
    pyinstaller --onefile --noconsole --name="DimensionShift" main.py

Готовый файл DimensionShift.exe появится в папке dist.
